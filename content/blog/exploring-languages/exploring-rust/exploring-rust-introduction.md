I'm looking at Rust first. I haven't done any memory management since sophomore year of undergrad (in C), so this should be a nice refresher on how to not leak memory.

Before I begin, what appeals to me about Rust?

- Static typing
- (Some) type inference
- No concept of `null`
- Possibility for performance increases (over languages I currently used)

I'll be reading ["The Book"](https://doc.rust-lang.org/book/) and working through the exercises.
